Netflix Clone


In this project I used the power of Flutter to create a web and android version of my own netflix clone. Flutter makes it really easy to create cross-platform apps and I believe it now more than ever before. With virtually the same code, with minor adjustments for different screen sizes we can create apps not only for android or ios, but also for other platforms such as Desktop, Televisions and even SmartWatches! This was the first project where I used Firebase as backend and I was really amazed by how convenient it is to use with flutter.


Login Screen

This is the first screen a new user will see. It will ask for user credentials to log into the app. The authentication process is carried out by Firebase authentication services. If user doesn't have an account, a link given below takes the user to Sign Up screen.

Home Screen

In the home screen, a user can find all the content uploaded on the app sorted by categories, as well as user's own watch history shown as "My List". If the user is an admin, they can see the Admin panel button just below the welcome message. This button is not shown to any user who is not an admin. I have shown the working of Admin panel in the descriptions below.

Video Play Screen

This screen is where the content will be played. It consists of a youtube video player package which requires URL of the video to be played. As we scroll down on the page, we see the content's description, the views of that content on the clone app and also a recommendation list based on the number of views. For the clone version, app only plays the trailers of content instead of originial episodes or movies.

"See All" Grid Screen

In front of each category on home screen, there is a button called "See All". This button takes us to the "See All" grid where all the content of that category is visible in a grid view instead of a list view. This screen is for the convenience of user.

Search Screen

This screen allows user to directly search content by name. User has to type full name of desired content.

Admin Panel

This screen will only be available to the admin. From here the admin will have the option to add new content or update/delete some existing content. If admin wants to update some existing content, they will have to click on update content button and then search the name of content. After clicking on the found content, it will take admin to update content screen where all the fields will be pre filled with existing data. Admin can make changes or delete that content easily.

Add New Content Screen

This screen will allow admin to add new content. The fields required are content name, content image URL, content video URL and content description. After filling these fields, admin can select the genre of content from dropdown list. Finally clicking on "Add now" button will add the content to database and take admin back to the home screen.

Profile Screen
